# Resume Analyzer

This project analyzes a resume and provides:

## Features
- Resume scoring (out of 10)
- Feedback on missing sections
- Final suggestion for improvement

## Sections Checked
- Name
- Skills
- Education
- Experience
- Projects
- Email

## How to Use
1. Paste your resume text
2. Run the program
3. Get score and feedback

## Tech Used
- Python
- Google Colab
- Gradio (for demo)
